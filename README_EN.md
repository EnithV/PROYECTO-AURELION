## Overview

**Aurelion Project** is an end-to-end data analysis and machine learning system designed to support business decision-making for a retail store.

The project is structured into **three sprints**, covering:

- Exploratory Data Analysis (EDA)
- Statistical analysis
- Machine Learning models (supervised and unsupervised)
- Interactive programs and automated documentation

All components are integrated through a **unified interactive Python program**.

---

## Project Structure

- **Sprint 1 – Data Analysis**
  - Exploratory data analysis
  - RFM customer segmentation
  - Executive reports
  - Interactive menu-based system

- **Sprint 2 – Machine Learning & Data Normalization**
  - Advanced data preprocessing
  - Regression, classification and clustering models
  - Inferential and prescriptive statistics
  - Automated generation of reports and visualizations

- **Sprint 3 – Machine Learning Fundamentals**
  - ML theory fundamentals
  - Model training and evaluation
  - Metrics comparison
  - Interactive demo
